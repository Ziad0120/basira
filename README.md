لتشغيل السيرفر (السيرفر مربوط بالفرونت اند) يجب توفر الآتي:
Node.js إصدار 18 أو أحدث
npm (يأتي مع Node)

ثم يجب عليك الانتقال داخل مجلد الباك اند 

cd /ai-project/backend

و تبدأ بالآتي ::


اوامر تحميل المكاتب : 

npm install -g edge-tts

و تتأكد من : 
edge-tts --help

و ايضا :
npm install express multer dotenv pdf-parse mammoth groq-sdk


ثم التشغيل : 
node server.js


هيشتغل على localhost:3000
